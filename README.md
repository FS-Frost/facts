# Facts

App to get random facts from [Random Useless Facts API](https://uselessfacts.jsph.pl/).

## Built with

-   React
-   TypeScript
-   Jest

## Use the app

https://fs-frost.github.io/facts/

## Run locally

```shell
yarn install
yarn start
```

Visit http://localhost:3000/facts

## Run tests

```shell
yarn run test
```
